# New Tab in Group

A Google Chrome Extension that allows the user to open a new tab in the current tab group using the following shortcuts:
- "windows": "Ctrl+Shift+Y",
- "mac": "Command+Shift+Y",
- "chromeos": "Ctrl+Shift+U",
- "linux": "Ctrl+Shift+J"
